Learning Feature Pyramids for Human Pose Estimation

发表于 2017-11-12 | 分类于 human pose estimation

Abstract

问题的难点：

  1. 当摄像机视角变话或者视角发生严重缩短，导致图像虚化 带来的人体骨架部位的尺度变化；
  2. pyramid methods经常在使用在inference阶段对图像进行尺度改变，在DCNNs中学习特征pyramids还未被开发

本篇文章重点：

  1. 设计 PRMs(Pyramid Residual Module) 来捕捉 DCNN 中尺度的变化，PRMs会从 不同尺度的输入特征 学习不同的 卷积核；
  2. 对于 multi－branch 网络，探索一个新的初始化参数的方法；
  3. 残差会随着深度的增加，损失会急剧增大，本文也提到一个高效的方法解决这个问题

Frame Work

PRM 提起feature －－－》stack hourglass network 获得人体骨架关键点

Revisiting Stacking Hourglass Network

Hourglass network 用于在 前向阶段 图片不同尺度的信息；bottom-up：subsampling 特征图，top-down:上采样特征图；在bottom－up 和 top－down两个阶段重复多次构成 “stacked hourglass“” 网络结构；

但是在原论文中，残差单元作为hourglass network的构建模块(积木)，只能捕捉一个scale下的可视化模式或者语义信息。本文使用的pyramid residual module作为 hourglass network的构建模块（积木）能够获取多尺度下的可视化模式或语义信息。

RMPE: Reginonal Multi-Person Pose Estimation

发表于 2017-10-26 | 分类于 human pose estimation

本文发表在 iccv 2017会议上，由上海交通大学 hao-shu fang, shuqin xie 和腾讯ailab yu-wing tai共同合作发表，针对 two-step human pose estimation method作出改进，并在mpii 和 coco 测试集上得到非常优异的结果，高于cmu_pose分数0.01；

Abstract

对于多人姿态估计研究领域中，研究方法有两种，一种是基于part-detection,另一种是two_step method，human_detection + single_person_pose_detection。其中在two_step method中，human_detection检测器的效果好坏对结果的影响很大。所以本文提出一种新的框架RMPE，应用于不准确的行人检测器，减少检测器检测行人误差带来的影响从而也能能得到更好的pose estimation结果。本文主要包括三个部分：

  1. Symmetric Spactial Transformer Network(SSTN): 对称空间变换网络
  2. Parametric Pose NoMaximum-Suppression(NMS): 带参数的非最大抑制
  3. Pose-Guided Proposals Generator(PGPG): 姿态导向的 候选框的提取

Related Work

Single Person Pose Estimation

这部分单个行人检测器仅仅在人的位置被准确定位时，效果很好；其中方法主要分为：传统方法，例如树模型、图模型；基于深度学习方法：DeepPose, 还有其它基于cnn和dnn的方法；

Multi Person Pose Estimation

Part-based Framework

该部分研究方法主要研究检测行人的关键点部位，再通过将不同关键点聚集成不同人。主流方法有：deepcut，cmurtpose，个人推崇先看最新得分最高的结果，目前我在使用cmurtpose做人体骨架关键点检测，在数据集上训练后得到结果，发现效果惊人的同时也发现还是存在很多问题的，对于这些问题值得进一步思考。

Two-step Framework

本文工作主要基于此。human detector + pose detector;
本文结合Faster-Rcnn＋基于cnn的方法SPPE的姿态估计比Insafutdinov使用的Faster-Rcnn+DeepCut在mpii数据集上测试的mAP51.0要高的多，为76.7。通过结合行人检测器和姿态估计算法的发展，two－step framework能够得到近一步的提升，本文也提出如何解决对于impeferct的检测器带来的问题从而最大化SPPE的效果。

建议初学这可以把文中提到的论文简要阅读并总结，了解该问题的研究历史。

Architecture

输入human_detector检测的human proposals，输入空间转换网络（STN）,用于提取质量更高更有意义的human proposal（STN前辈提出的网络用于提取更加有效的区域），同时输入SPPE 和 Paraller SPPE，上部分用于预测human pose，下半部分用固定参数的SPPE用于训练，计算loss，保证训练时全局的稳定性，同时当出现检测区域检测人体姿态效果很差时，能够回传一个大的loss，告诉网络该proposal不适合做人体姿态估计；上部分SPPE的结果经过SDTN反变换为原图像空间，再经过一个带参数的nms得到最后的人体姿态估计结果。

Symmetric STN and Parallel SPPE

STN and SPPE

空间变化网络在自动选择感兴趣区域时有非常好的表现，所以本文使用空间变换网络用于提取高质量重要的人体检测提要框。
STN: 2D
SDTN: 得到的结果经过反变换为原图像空间

在训练中,STN和SPPE一起训练。

Parallel SPPE

为了更好的通过STN提取到有效的区域，增加Parallel SPPE直接对extracting human proposals进行测试，与真实的label进行比对计算loss，如果extracting human proposal并不是把物体包含在提要框中心，则会得到很差的pose estimation结果，从而得到非常大的loss。所以该网络的目的是用来回传与基于中心区域的groundtruth pose的loss，这样能够帮助STN关注到提取正确的区域，提高提取感兴趣的效果。该网络的参数固定，只用于training phae，向STN module 回传loss

Parametric Pose NMS

基于数据驱动的参数NMS, 通过一定数据集训练得到不同部位的最大化抑制参数，而非手工设定，一旦学习到，就固定。

Pose-guided Proposals Generator

数据增强的方法。通过分析检测器的对于不同姿态检测结果与真实结果的误差分布，利用数据增强模拟检测器的误差，让后续的STN和SPPE适应检测器带来的误差，从而能够学习到更好的效果；

MOT : Multi-Object Tracking

发表于 2017-10-24 | 分类于 MOT

abstract

寻找各项比赛，需要考虑fps，accuracy，使用什么样的方法（pdf，code）,各项比赛的evaluation
寻找有效的dataset，跟踪目标主要是：车，人脸，人，目标是能够达到在跟踪7～10人能够达到30fps；

为什么深度学习对于跟踪目标不好：跟踪10个苹果，而在现实场景中，跟踪物 人和车的相似度经常很相似

mot challenge（2016）

dataset

2015 年数据集的缺陷：

  • 标注协议不一致因为有些标注是直接从其它已经标注好的数据集中选择出来的；
  • 人群密度对于 训练集和测试集没有平衡
  • 有一些数据集过于简单（PETS09-S2L1），方法容易在这上 overfitting，所以这是不合理的训练集
  • 给定的检测方法在数据集上表现不够

相比于2015 年数据集的优势：

  • 给定14个数据集序列，更加的密集，从不同视角，运动相机，不同天气条件
  • 另外这些标注都被严格检验两遍，所所以标注有很高的准确性
  • 标注的类别也丰富，人，车，遮挡物，干扰物等，能够更加方便来计算被遮挡的程度
