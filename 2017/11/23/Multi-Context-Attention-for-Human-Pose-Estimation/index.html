Multi-Context Attention for Human Pose Estimation

发表于 2017-11-23 | 分类于 human pose estimation

Abstract

本篇论文提出将multi-context attention的机制集成到卷积神经网络中，用于人体骨架关键点检测。主要使用 stacked hourglass网络从不同分辨率和不同语义的特征中产生attention map(关注图)。

本文的主要贡献：

  • 提出使用 视觉注意机制（visual attention mechanism）学习和推断上下文表示，关注感兴趣的区域；相比于传统注意机制模型中使用空间softmax归一化，本文使用CRF对相邻的关节点空间相似性进行建模；
  • 使用多重上下文注意机制（multi-context attention）使模型更加鲁棒和准确，本文使用在每一个 hourglass中使用multi-resolution ，不同的hourglass之间实现multi-semantic，最后使用 hierarchical visual attention 来聚焦于局部区域看的更清楚；
  • 新设计一个通用的 hourglass residual unit, 即在单元内部扩展一个大的图像感知域的分支，因此不同尺度的特征能够被很好的学习。

Framework

  • Baseline Network: 8-stack hourglass network
  • Nested Hourglass Network: 使用HRU结合多分辨率下的特征
  • Multi-Resolution Attention: 在每个hourglass中，多分辨率的关注图从不同尺度的图像中产生
  • Multi-Semantic Attention: 低层stack关注具体表象，高层stack编码全局的表现；所以从不同stacks中产生的attention maps也学习到不同语义的信息
  • Hierarchical Attention Mechanism: 1-4层stack，使用2个全局关注图来编码整个行人的特征，在高层，5-8层我们设计精细化的主意机制到更加局部的部位

Attention Mechansim

输入feature map，生成多分辨率的attention map, 生成$h^{att}{1}$, 传统的attention mp 直接使用在由它产生的feature map 上，但是用attention map调整的feature 有很多的0，同时因为一个stack有多个调整的特征图，使用back-probagation较为困难，所以为了保证训练的效率，我们从不同的分辨率的特征图产生多个attention maps，再统一将它们应用在随后的feature； 除了多分辨率的attention，最后对产生$h^{att}{1}$ 再使用一个 attention map 产生$h^{att}{2}$
