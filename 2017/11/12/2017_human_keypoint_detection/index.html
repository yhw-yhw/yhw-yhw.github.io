#2017 Human Keypoint Detection

Multi-person keypoint detection

2017 COCO keypoint detection

face++

  1. mask-rcnn to detect person
  2. single-person skeleton model based on Resnet with U-shape side branch to integrate high resolution information from the earlier layers and semantic information from the deeper layers.
  3. only using the provided training set

BEIHANGU

  1. top-down method
  2. first stage: mask-rcnn to detect person
  3. second stage: using sppe.
  4. rm false annotated data
  5. use both detection and keypoint score for instance re-scoring and perform box-nms and oks-nms based on it.
  6. both are single model, no model ensembles
  7. using aic dataset to improve 1.5 map

BANGBANGREN

  1. two-stage system
  2. first stage: faster rcnn with deformable version of resnet101 on MS-COCO trainval2017 Multi-scale testing and soft-NMS
  3. second stage: single person pose estimation, train a resnet-101 based model to predict heatmap and offset fields of keypoints for the person in the detected bounding box
  4. The performance is boosted by OKS-based NMS in post-processing
  5. single person pose model trained COCO MS-trainval2017 and extra dataset; and using an ensemble of seven models to obtain an average precision of 0.728 on the test-dev set and 0.706 on the test-challenge set.

G-RMI

  1. Associative Embedding:
    End-to-End Learning for Joint Detection and Grouping
  2. Towards Accurate Multi-person Pose Estimation in the Wild

FAIR

  1. mask-rccn
  2. feature pyramid networks
  3. resnext
  4. semi-supervised learning paradigm

paper:

  1. Aggregated Residual Transformations for Deep Neural Networks
  2. Feature Pyramid Networks for Object Detection

SJTU

  1. get 69.7 map on test_dev dataset by fusing three model

2016 COCO keypoint detection

CMU_rtpose

  1. bottom_up method
  2. first, detect all keypoints and all limb vectors
  3. second, accumulate different keypoints to different person
