目前观看测试图像 1500 张；

  • 测试图像中出现 边连接 （0,0,0）；出现连接（width, 0, 0）当测试时，图像输入三个 scale，1，0.85, 0.7时会出现边角点误检测的问题；

  • 当同一个人体间检测关键点重合，会导致同一个人边的错连


  • 当检测关键点 缺失未被检测，容易发生不同人体间关键点的连接



  • 体操运动员 腿部关键点未检测

  • 目标身穿肥大衣服，整体一致性很强时，很难检测关键点

  • 对于目标 背面的 腿部关键检测容易检测 相反

  • 手腕未被检测

  • 腿部关键点检测错误 - 检测相反

  • 对于图像中太小的行人无法检测

  • 对于膝盖部位，太过平坦区域检测较弱

  • 训练迭代次数不够，不断训练能够加强网络对关键点检测的准确性和recall
    左图为 22w次在 mpii 数据集上训练的网络测试结果，右图为 23w次在 aichallenger 数据集上训练的结果；

  • 当发生 rhip 和 lhip缺失时， 容易造成 上半身的关键点与 下半身关键点 连接不上；
    左图为 连接阈值 0.01，右图连接阈值为 0.2
